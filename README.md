# ATHRAV_BOT

ATHRAV_BOT is a compact quadruped robot built around the ESP32 microcontroller. The robot features four articulated legs, an OLED display for system information, and wireless remote-control capability for movement and operation.

Designed as a robotics learning and development platform, ATHRAV_BOT combines embedded systems, mechanical design, and wireless communication in a lightweight and modular structure. The quadruped architecture provides stability and mobility while allowing future expansion with sensors, computer vision, and AI-based features.

## Features

* Four-legged (Quadruped) robotic platform
* ESP32-based control system
* OLED display for status and information
* Wireless remote-control operation
* Compact and lightweight design
* Multiple degrees of freedom for leg movement
* Expandable hardware architecture
* Suitable for robotics education and experimentation

## Applications

* Robotics Learning
* Embedded Systems Projects
* Motion Control Research
* STEM Education
* Autonomous Robotics Development
* AI and Sensor Integration Experiments

## Future Scope

ATHRAV_BOT can be further enhanced with obstacle detection, camera modules, voice control, computer vision, autonomous navigation, machine learning capabilities, and mobile application integration.
